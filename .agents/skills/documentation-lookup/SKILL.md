# Documentation Lookup

Use this skill whenever the request depends on current library behavior.

## Triggers

- Next.js, React, Tailwind, Radix, SWR, Zustand, or MapLibre API questions
- Setup or configuration tasks
- Version-specific or recently changed framework behavior

## Workflow

1. Resolve the library against an official or primary documentation source.
2. Query the docs with the user's exact question.
3. Answer from the docs, not from memory.
4. If docs are unavailable, say that clearly and note the uncertainty.

## Rule

Do not guess on unstable APIs, especially for Next.js 16.
